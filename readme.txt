プラグインの動作:
config.ymlのpermittedWorlds以下にあるワールド以外にログインした場合に、
teleportDestが指し示し場所へとテレポートさせます。
teleportDestのworldキーはテレポートさせるワールドの名前、
その下のx, y, zは座標を入力して下さい。もしも座標が指定されなかった場合には
自動的にスポーン地点へとテレポートされるようになっています。


このプラグインは修正BSDライセンスによって提供されるものとします。
